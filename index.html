Audio Player Editor for Genially

How to use:

  1. Customize player settings below
  2. Check the preview
  3. Click "Get Code" to copy embed code
  4. Paste code directly into Genially HTML widget

Requirements: Create a playlist.json file in your GitHub repo with audio track list. Example structure:

{
  "tracks": [
    {
      "title": "Track 1",
      "artist": "Artist 1",
      "file": "track1.mp3",
      "image": "cover1.jpg"
    }
  ]
}
